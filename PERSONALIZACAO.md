# Personalizacao do site

Use este checklist antes de publicar o site da Dra. Mayara.

## Dados principais

Atualize os mesmos dados em:

- `site/index.html`
- `site/versao-referencia/index.html`

Campos para revisar:

- Nome profissional.
- Especialidade.
- CRM.
- RQE.
- Cidade e estado.
- Endereco do consultorio.
- Modalidade de atendimento: presencial, online ou ambas.
- Texto de apresentacao.
- Areas de cuidado.
- Duvidas frequentes.

## WhatsApp

Procure por:

```txt
5500000000000
```

Substitua pelo telefone real no formato internacional, sem espacos:

```txt
55DDDNÚMERO
```

Exemplo ficticio:

```txt
5511999999999
```

Procure tambem por:

```txt
(00) 00000-0000
```

Troque pelo telefone formatado para leitura.

## Instagram

Procure por:

```html
href="#"
```

No link de Instagram, troque pelo perfil correto, por exemplo:

```html
href="https://www.instagram.com/perfil-da-dra"
```

## Localizacao

No botao e no rodape, troque `#localizacao` ou o endereco placeholder pelo link real do Google Maps.

## Imagens

A imagem atual fica em:

```txt
site/images/consultorio-mayara-cardoso.png
```

Para trocar:

1. Coloque a nova imagem em `site/images/`.
2. Use um nome simples, sem espacos, por exemplo `dra-mayara-cardoso.jpg`.
3. Atualize os caminhos nos dois HTMLs.

Locais para procurar:

```txt
consultorio-mayara-cardoso.png
```

Recomendacoes:

- Use uma foto horizontal para o topo do site.
- Use imagem com boa luz e alta resolucao.
- Evite fotos muito escuras ou com texto embutido.
- Comprima a imagem antes de publicar, se ela ficar muito pesada.

## Favicon

O favicon fica em:

```txt
site/favicon.svg
```

Voce pode substituir por um arquivo da identidade visual da Dra. Mayara mantendo o mesmo nome.

## Versoes do site

- Versao principal: `site/index.html`
- Versao alternativa: `site/versao-referencia/index.html`

A versao alternativa continua disponivel para a Dra. Mayara comparar a identidade visual.

O rodape inclui um aviso institucional:

```txt

```

Mantenha ou ajuste esse texto conforme a orientacao profissional da medica.
